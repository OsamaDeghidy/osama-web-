# ملخص مشروع OSERA Web

## 1. محتوى المشروع (What's in it?)
المشروع عبارة عن تطبيق ويب متكامل (Full-Stack) مبني باستخدام:
- **الواجهة الأمامية (Frontend)**: React مع Vite و TailwindCSS و Framer Motion (للحركات).
- **الخادم (Backend)**: Express.js مع TypeScript.
- **الذكاء الاصطناعي**: دمج مع واجهة برمجة تطبيقات Gemini (`@google/genai`).

**الميزات الرئيسية**:
- مسار المحادثة (`/api/chat`): مساعد ذكاء اصطناعي يمثل مؤسس الشركة (Osama Esmael) كخبير في هندسة النظم وتطوير Django.
- مسار المخططات (`/api/blueprint`): أداة لتوليد مخططات تقنية للمشاريع بناءً على المتطلبات، الميزانية، والجدول الزمني.
- مسار التحسين (`/api/django-optimize`): أداة لتقييم وتحسين أكواد Python/Django برمجياً وأمنياً.

## 2. التحسينات الممكنة (Possible Improvements)
- **الأمان**: إضافة تحديد معدل الطلبات (Rate Limiting) لمنع استنزاف حصة الـ API الخاصة بـ Gemini.
- **هيكلة الكود**: فصل المسارات (Routes) ومتحكمات الـ (Controllers) في ملفات مستقلة بدلاً من وضع كل شيء في `server.ts`.
- **التحقق من البيانات (Validation)**: استخدام مكتبة مثل Zod للتحقق من صحة المدخلات قبل إرسالها إلى Gemini.
- **معالجة الأخطاء**: توفير رسائل خطأ أكثر تفصيلاً للواجهة الأمامية وتسجيل الأخطاء (Logging) باستخدام مكتبة متخصصة مثل Winston بدلاً من `console.error`.
- **الاختبارات**: كتابة اختبارات (Unit Tests) للواجهة الأمامية والخلفية لضمان استقرار التطبيق.

## 3. كيفية التشغيل (How to run)
1. تأكد من تثبيت **Node.js** على جهازك.
2. افتح موجه الأوامر (Terminal) في مجلد المشروع ونفذ الأمر:
   `npm install`
3. قم بإنشاء ملف `.env` (أو استخدم `.env.local` كما هو مذكور في README) وأضف مفتاح API الخاص بك:
   `GEMINI_API_KEY=your_api_key_here`
4. لتشغيل خادم التطوير، نفذ الأمر:
   `npm run dev`
5. التطبيق سيعمل على الرابط: `http://localhost:3000`
